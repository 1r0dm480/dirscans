# Usage
```
USAGE : ./dirscans.sh -u [url]
E.g : ./dirscans.sh -u http://domain.com
	OPTIONS :
	-u (url) **required**
	-d (Custom Wordlist: default [dict/dict.txt]) **optional**
	-t (Thread: default [50]) **optional**
	-o (Output) **optional**
```
